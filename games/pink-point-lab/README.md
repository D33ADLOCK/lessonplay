# Pink Point Lab

Guided ChemQuest Lab template game about phenolphthalein indicator behavior:
colorless in acid or neutral solution, pink in base.

## Run

```bash
npm run dev --workspace pink-point-lab
```

## Validate

```bash
npm run typecheck --workspace pink-point-lab
npm test --workspace pink-point-lab
npm run build --workspace pink-point-lab
```
